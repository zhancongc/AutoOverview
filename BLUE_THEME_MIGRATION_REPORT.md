## 🎯 设计原则

### 为什么选择蓝色？

1. **专业感**: 蓝色在西方文化中代表专业、信任、稳定
2. **学术属性**: 蓝色常用于学术、科研类产品
3. **SaaS 标准**: 大多数海外 SaaS 产品使用蓝色（Stripe、GitHub、Notion）
4. **色彩心理学**: 
   - 蓝色：冷静、理性、可靠
   - 红色：紧急、警告、热情（更适合国内市场）

### 保留红色的情况

中文版产品仍使用红色系：
- `SimpleApp.css` - 中文主页
- `ProfilePage.css` - 中文个人中心
- `ReviewPage.css` - 中文综述页面
- `ReviewViewer.css` - 中文综述查看器

**原因**: 红色在中国文化中代表喜庆、热情、好运

## 📝 技术细节

### CSS 变量命名

```css
/* 之前 */
--academic-red: #D63031;
--academic-red-dark: #B71C1C;

/* 现在 */
--academic-blue: #0988E3;
--academic-blue-dark: #0770BD;
```

### 替换命令

```bash
# CSS 变量替换
sed 's/var(--academic-red)/var(--academic-blue)/g'

# 颜色值替换
sed 's/#D63031/#0988E3/g'
sed 's/rgba(214, 48, 49/rgba(9, 136, 227/g'
```
